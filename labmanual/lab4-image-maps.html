2. Rectangle (rect) Image Map

Click on the rectangular colored regions. Each rectangle is defined using shape="rect".

Syntax:

<area shape="rect" coords="x1,y1,x2,y2" href="#" alt="description">

coords: x1,y1 (top-left corner), x2,y2 (bottom-right corner)

3. Polygon (poly) Image Map

Click on the polygon shapes (triangle, pentagon, star). Each shape is defined using shape="poly".

Syntax:

<area shape="poly" coords="x1,y1,x2,y2,x3,y3,..." href="#" alt="description">

coords: x,y coordinates of each vertex of the polygon

4. Mixed Image Map

This image combines all three types: circles, rectangles, and polygons in a single map. Click on any shape!

Mixed Map Explanation:

A single image can have multiple area tags with different shapes. This is useful for complex clickable regions like website navigation, interactive diagrams, or game interfaces.

Image Map Structure:

  1. <img> tag with usemap="#mapname" attribute
  2. <map> tag with name="mapname" attribute
  3. <area> tags inside the map defining clickable regions

Shape Types:

  • shape="circle" - Circular regions
  • shape="rect" - Rectangular regions
  • shape="poly" - Polygon regions (any number of sides)
  • shape="default" - Entire remaining area (not shown in demo)

Use Cases:

  • Interactive diagrams and infographics
  • Geographic maps with clickable regions
  • Dashboard navigation
  • Product showcases with clickable features
  • Educational materials and tutorials
