Lab Program 3: CSS Styling Methods

This program demonstrates three different ways to apply CSS styles in HTML:

  1. Inline Styles
  2. Internal Style Sheet
  3. External Style Sheet

1. Inline Style Example

This section uses inline styles applied directly to HTML elements using the style attribute.

Inline styles have the highest priority and override both internal and external styles.

2. Internal Style Sheet Example

This section uses internal styles defined in the <style> tag within the <head> section of the HTML document.

Internal styles apply to the entire page and are useful when you want to style a single HTML document without affecting other pages.

Features of Internal Styles:

  • Defined within <style> tags in the <head> section
  • Apply to the entire HTML document
  • Override external styles but are overridden by inline styles
  • Useful for single-page styling
  • Can use CSS selectors, classes, and IDs

3. External Style Sheet Example

This section uses external styles defined in a separate CSS file (lab3-external-styles.css) linked to this HTML document.

Advantages of External Style Sheets:

  • Separation of content (HTML) and presentation (CSS)
  • Reusable across multiple HTML pages
  • Easier to maintain and update
  • Reduces page load time through browser caching
  • Keeps HTML code clean and organized

Comparison of CSS Styling Methods

Method Syntax Priority Use Case
Inline <tag style="property: value;"> Highest Quick specific changes
Internal <style> in <head> Medium Single page styling
External <link rel="stylesheet" href="file.css"> Lowest Multiple pages, best practice
