Summary

Heading tags: h1, h2, h3, h4, h5, h6

Important attributes:

  • id - unique identifier
  • class - for CSS
  • style - inline CSS
  • title - tooltip

Link attributes:

  • href - where the link goes
  • target - where to open (_blank for new tab)
  • title - tooltip
  • download - to download file
